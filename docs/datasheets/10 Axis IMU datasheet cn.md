# 10轴IMU数据手册

## 介绍

版本编码: O1252V10

![10 Axis IMU 渲染图正面](imgs/O1252V10-B1%20ST0341%2010-Axis%20IMU%20渲染图正面.png)
![10 Axis IMU 渲染图背面](imgs/O1252V10-B1%20ST0341%2010-Axis%20IMU%20渲染图背面.png)
10 轴 IMU 模块是一个集成了 3 轴加速度传感器、3 轴陀螺仪传感器和 3 轴磁力传感器和气压计的模块。模块使用I2C通信, 可以修改部分芯片的I2C地址. 板载LED灯, 可以指示电源状态.

模块主要芯片包括:
- SH3001 3 轴加速度传感器和 3 轴陀螺仪传感器
- QMC6310 3 轴磁力传感器
- SPL06-001 气压计

规格参数:
- 工作电压: 2.5-3.6V
- 工作电流: 4mA
- 3 轴加速度传感器
- 3 轴陀螺仪传感器
- 3 轴磁力传感器
- 气压计
- I2C 通信
- 可修改部分芯片的I2C地址
- 板载LED灯, 可以指示电源状态

## 引脚定义

板子包含两个I2C接口, 都是SH1.0 4P结构, 兼容qwIIC, Stemma QT, Modulino等接口. 两个接口并联, 可以用来扩展到其他I2C设备.

| 引脚 | 功能 |
| --- | --- |
| SCL | I2C 时钟线 |
| SDA | I2C 数据线 |
| VCC | 电源输入, 2.5-3.6V |
| GND | 接地 |

## 地址定义

### SH3001 I2C 地址

SH3001 默认地址是0x36, 可以通过修改连接方式, 切换到0x37.

板子上左边标着0x3的3P焊盘跳线是SH3001的I2C地址跳线. 默认链接左边6, 就是0x36. 可以把左边的连接线切断, 然后用烙铁焊锡, 链接右边7的位置, 就可以把地址切换到0x37.

### QMC6310 I2C 地址

QMC6310 地址是固定的0x1C 或者0x3C, 根据芯片不同会有不同, 不可手动选择切换

### SPL06-001 I2C 地址

SPL06-001 默认地址是0x76, 可以通过修改连接方式, 切换到0x77.

板子上右边标着0x7的3P焊盘跳线是SPL06-001的I2C地址跳线. 默认链接左边6, 就是0x76. 可以把左边的连接线切断, 然后用烙铁焊锡, 链接右边7的位置, 就可以把地址切换到0x77.
